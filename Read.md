# Projet HTML & CSS – Description des choix de conception

## Description générale

Lors de ce projet, j’ai structuré mon site en utilisant des **balises sémantiques** afin d’assurer une organisation claire et logique du contenu.

Le site se compose :
- d’une **barre de navigation**,
- de **trois sections principales** : Hero, Produits et Contact,
- ainsi que d’un **footer**.

Chaque section contient différentes divisions incluant des **images**, des **titres et sous-titres**, un **formulaire**, ainsi qu’une **barre de défilement**, permettant une meilleure structuration et lisibilité du contenu.

## Choix de design

Pour le design, j’ai opté pour un style **simple et chic**, basé sur une palette de couleurs **noir et doré**, en accord avec le thème du site et visant à offrir une interface élégante et moderne.

## Responsivité

Enfin, le site a été rendu **responsive** afin de s’adapter correctement aux différentes tailles d’écran :
- Desktop (L)
- Tablette (M)
- Mobile (S)

Cela permet d’assurer une expérience utilisateur optimale sur tous les supports.
